Jump Your Lazy Ass To The Flag, Yo!

Created by Iwan Gabovitch <qubodup@gmail.com>.

Started during Berlin Mini Jam 2012-03-24 <http://berlinminijam.blogspot.com/>.

'Play' it on <http://qubodup.github.com/jylattfy/>!

developed on arch linux.
code written using vim.
tested using chrome.
images made using gimp.
sounds generated using sfxr.
pappapia recorded using an htc wildfire aka buzz smartphone and edited in audacity.

steps.wav created using sox:
sox step1.wav tmp1.wav pad 0 .1
sox step2.wav tmp2.wav pad 0 .1
sox tmp* steps.wav
rm tmp*
